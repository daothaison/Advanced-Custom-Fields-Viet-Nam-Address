== Description ==

Plugin này giúp các bạn thêm mục chọn [đơn vị hành chính của Việt Nam](http://levantoan.com/mysql-don-vi-hanh-chinh-cua-viet-nam-chinh-xac-toi-xaphuongthi-tran-moi-nhat/).
Lựa chọn địa chỉ như tỉnh/thành phố; quận/huyện; xã/phường/thị trấn. Rất phù hợp với các bạn làm web bất động sản, hoặc vấn đề liên quan tới địa chỉ cụ thể và chính xác

Thông tin chi tiết và thắc mắc hãy comment [tại đây](http://levantoan.com/acf-viet-nam-address) hoặc ngay trên wp.org nhé

= Compatibility =

This ACF field type is compatible with:
* ACF 5
* ACF 4

== Installation ==

1. Copy the `acf-viet-nam-address` folder into your `wp-content/plugins` folder
2. Activate the ACF Viet Nam Address plugin via the plugins admin page
3. Create a new field via ACF and select the Viet Nam Address type
4. Please refer to the description for more info regarding the field type settings
